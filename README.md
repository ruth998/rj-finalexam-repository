Project Title:Inventory and Purchase Order Management System. 

 Name: Ruth Shalina Jacquet
 
 Course Section: Computer Programming 
 
This project is a Python-based console application designed to simulate a small business inventory system that manages products, vendors, and purchase orders while tracking stock levels and generating reports. It includes features such as adding, viewing, editing, and deactivating products, searching products by ID or name, monitoring low stock items, managing vendors, creating and tracking purchase orders, updating inventory based on shipments, and generating inventory reports. The required files for this project include `main.py` as the main entry point, `models.py` for the Product, Vendor, and PurchaseOrder classes, `inventory_manager.py` for product operations, `file_manager.py` for saving and loading data, `reports.py` for generating reports, and JSON files used for data storage. To run the program, ensure Python 3 is installed, place all files in the same folder, open a terminal in that folder, and run `python main.py`, then follow the menu instructions. Data is stored using JSON files where each object is converted into a dictionary using a `to_dict()` method, saved to files, and loaded back into objects when the program starts to ensure persistence. Extra features include low stock alerts, product search by name, inventory value reporting, and purchase order tracking with status updates such as Pending and Completed, making the system more realistic and efficient.
